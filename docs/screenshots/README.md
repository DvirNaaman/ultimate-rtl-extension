# Screenshots

This folder holds the screenshots referenced from the main `README.md`.

Drop the PNG files in this folder using exactly these filenames:

| Filename | What it should show |
|---|---|
| `claude.png` | claude.ai with a Hebrew + English mixed message, RTL working |
| `chatgpt.png` | chatgpt.com with a Hebrew + English mixed response |
| `gemini.png` | gemini.google.com with a Hebrew + English mixed response |
| `perplexity.png` | perplexity.ai with a Hebrew query and answer |
| `copilot.png` | copilot.microsoft.com with a Hebrew + English message |
| `deepseek.png` | chat.deepseek.com with a Hebrew + English mixed response |
| `grok.png` | grok.com with a Hebrew + English mixed response |
| `popup.png` | The extension popup with the "Enable RTL" toggle |

Recommended size: about **1200×800** PNG. The Chrome Web Store listing
also accepts these (cropped/resized to 1280×800 if needed).

You can use the screenshots captured during this project's development
session — they show exactly the supported workflow.
